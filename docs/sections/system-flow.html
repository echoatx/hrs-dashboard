System Flow

Coordinated Entry Overview

The set of figures below show measures on how many people go through Coordinated Entry (CE) and how long people take to go through the CE process for each measurement during a given month. All time measures reported are median (the 50th percentile) number of days rather than averages, to account for outliers that would otherwise skew averages.

For more information on all these measures, see the More Info page.

Note

The Days from CA to Referral metric on the “System Flow Time” chart is measuring referrals sent in each given month for all CE participating projects, and the days from CA to Referral is calculated from the CA that resulted in the referral, which is the most recent assessment.

Coordinated Entry Flow Details

The following figures show how long people take to go through the CE process, broken down by race/ethnicity, household type, age group, and project type. All time measures reported are median (the 50th percentile) number of days rather than averages, to account for outliers that would otherwise skew averages.

For the race/ethnicity plots, we only include numbers for Black, non-Hispanic White, and Hispanic/Latino clients, because the numbers for other race/ethnicity groups (often fewer than 10 per month) are too low and possibly misleading.

For more information on all these measures, see the More Info page.
